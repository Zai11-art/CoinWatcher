# CoinWatcher
A social media web app where you can check, monitor, and share cryptocurrency prices to friends and followers. 

Technologies used: 
* Mongodb
* Express
* Reactjs
* Nodejs
* Typescript
* Tailwindcss
* React-dropzone
* Formik
* React-chartjs-2
* Redux
* React-toastify
* S3 bucket
* axios
